
ATA
===
    Advanced Technology Attachment

IDE
===
    Integrated Drive Electronics
    集成驱动电子设备

    IDE 是一种计算机系统接口，主要用于
    硬盘和 CD-ROM，本意为 "把控制器与
    盘体集成在一起的硬盘"

ATA vs IDE
==========
    ATA 是一个控制器技术。
    IDE 是一个匹配 ATA 的磁盘驱动器技术。
    但是两个术语经常可以互换。


PATA
====
    Parallel ATA
    并行高技术配置

    就是原有的 ATA 改名而来，以区别 STAT。
    (= ATA)
    (= IDE)

SCSI
====
    Small Computer System Interface


SSD
===
    Solid State Drive
    固态硬盘


串口 vs 并口
============

    串行接口：是指数据一位一位地顺序传送。其
    特点是通信线路简单，只要一对传输线就可以
    实现双向通信。

    并行接口：指采用并行传输方式来传输数据的
    接口标准。SCSI 和 IDE 都是并行接口。

    并口硬盘指 IDE 硬盘。
    串口硬盘指 STAT 硬盘。


AHCI
====
    
    Advanced Host Controller Interface
    高级主机控制器接口
    
    可以让 STAT 存储设备激活高级 SATA 功能。
